# Juego del Gato (Tic-Tac-Toe) en Java

Juego clásico del **Gato (Tres en Raya)** desarrollado en **Java**, ejecutado desde consola.  
Proyecto enfocado en practicar **POO, control de turnos, validaciones y lógica de juego**.

---

## Características
- Juego para **2 jugadores**
- Control automático de turnos (X / O)
- Validación de:
  - Posiciones fuera del tablero
  - Casillas ocupadas
- Detección de:
  - Victoria por filas, columnas y diagonales
  - Empate
- Código refactorizado en métodos

---

## Tecnologías usadas
- Java SE
- Programación Orientada a Objetos
- Consola (CLI)
